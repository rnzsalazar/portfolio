Work Projects

Production ETL Pipeline

Developed a backend pipeline that reads `.dbf` files containing incoming work orders. Used Python and Pandas to clean and reformat the data, then automated updates via cron jobs every 15 minutes. The data was pushed to a Firebase Realtime Database for live access.

Order Display Dashboard

Created a Vue.js web application that visualized production order data using a responsive DataTable. Designed for large TV displays, it auto-paginated every 15 seconds and looped through the entries. Rows changed color based on the due date value (yellow if the due date was within 2–4 days from current date, and red if the due date was within a day or already passed). The table showed the work order, part number, quantity, and the due date.

Visitor Sign-In Android App

Designed a visitor management app where guests filled out a form on an android tablet, took a photo, and printed an ID badge for them. Badges featured a QR code, name, time in, and the ID number was based on the timestamp. Visitors could sign out manually or by scanning their badge's qr code. The app also allowed exports of visitor logs to Excel for logging purposes.

Product PDF Label Generator

Built a Java application using iText7 to create part-specific labels compatible with Avery templates. Each label included the company banner, QR code, part number, serial number, and part description. The user would select the label they wanted based on the category (lamp, bulb, harness/adapter, or dimmer), then they put in the part number and serial number, the rest would be pulled from the database.

Media Server Deployment with Proxmox
